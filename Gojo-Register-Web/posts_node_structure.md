## Field Details

### Posts Node Fields
- **postId**: Unique identifier generated by backend
- **message**: Post text content from `postText` state
- **postUrl**: Media URL from `postMedia` upload (null if no media)
- **financeId**: Finance user ID who created the post (`finance.userId`)
- **financeName**: Finance display name (`finance.name`)
- **financeProfile**: Finance profile image URL (`finance.profileImage`)
- **time**: ISO timestamp when post was created
- **likeCount**: Total number of likes (calculated from likes object)
- **likes**: Object with user IDs as keys and boolean values
- **seenBy**: Object tracking which users have seen the post

### Users Node Fields
- **userId**: Firebase unique key
- **username**: User's username
- **name**: User's display name
- **profileImage**: Profile picture URL
- **role**: User role ("finance", "student", "teacher")

### Teachers Node Fields
- **userId**: Reference to Users node
- **name**: Teacher's display name
- **profileImage**: Teacher's profile image

### Students Node Fields
- **userId**: Reference to Users node
- **name**: Student's display name
- **profileImage**: Student's profile image
- **grade**: Student's grade/class

### Parents Node Fields
- **userId**: Reference to Users node
- **name**: Parent's display name
- **profileImage**: Parent's profile image
- **studentId**: Reference to student's userId

### Chats Node Fields
- **messages**: Array of message objects
- **senderId**: ID of message sender
- **receiverId**: ID of message receiver
- **message**: Message text content
- **time**: ISO timestamp
- **seen**: Boolean indicating if message was read
